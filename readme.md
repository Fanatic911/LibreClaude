# LibreClaude Project Presentation

LibreClaude is an open-source, unrestricted AI ecosystem designed to provide developers, creators, and daily users with top-tier artificial intelligence tools completely free of charge. Built with the philosophy of unrestricted access to knowledge, LibreClaude removes the barriers of traditional AI platforms by offering a diverse selection of models tailored for specialized tasks.

## Key Features
* **100% Free & Unlimited:** No hidden fees, no premium tiers, and absolutely no token limits. You can code, write, and process large datasets without worrying about quotas.
* **Multi-Model Architecture:** Instead of relying on a single network, LibreClaude integrates a variety of specialized models to ensure maximum efficiency depending on your needs.

## Supported Models

### Groq Infrastructure Models
* **Llama 3.3 70B** (`llama-3.3-70b-versatile`) - Powerful and versatile general-purpose model.
* **Llama 3.1 8B** (`llama-3.1-8b-instant`) - Ultra-fast and instant responses.
* **GPT-OSS 20B** (`openai/gpt-oss-20b`) - Lightweight and efficient open-source architecture.
* **GPT-OSS 120B** (`openai/gpt-oss-120b`) - Heavy-duty model tailored for advanced reasoning and coding.

### GitHub Infrastructure Models
* **Codestral** (`mistral-ai/Codestral-2501`) - Deeply specialized in coding, syntax generation, and debugging.
* **GPT-4o** (`openai/gpt-4o`) - Highly versatile, polyvalent model balancing speed and intelligence.
* **GPT-4o mini** (`openai/gpt-4o-mini`) - Fast, lightweight version for quick conversational interactions.
* **GPT-4.1** (`openai/gpt-4.1`) - Next-generation model built for advanced coding workflows.
* **GPT-4.1 mini** (`openai/gpt-4.1-mini`) - High-speed model optimized for rapid code generation.
* **o1-mini** (`openai/o1-mini`) - Advanced reasoning model tailored for complex problem-solving.
* **DeepSeek R1** (`deepseek/DeepSeek-R1`) - State-of-the-art reasoning and multi-step logic architecture.
* **DeepSeek V3** (`deepseek/DeepSeek-V3-0324`) - High-performance model focused on clean code generation.
* **Llama 3.3 70B Instruct** (`meta/Llama-3.3-70B-Instruct`) - Versatile, instruction-tuned model.
* **Llama 3.1 405B** (`meta/Meta-Llama-3.1-405B-Instruct`) - Massive-scale model built for deep comprehension and complex datasets.
* **Mistral Large** (`mistral-ai/Mistral-Large-2411`) - Flagship European model, highly polyvalent and robust.
* **Phi-4** (`microsoft/Phi-4`) - Lightweight, highly efficient model with strong reasoning capabilities.
