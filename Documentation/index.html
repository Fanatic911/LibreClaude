GETTING STARTED

Overview

LibreClaude is a free, open-source AI platform that brings together the best large language models and image generation tools into a single, unrestricted interface — with no subscriptions, no token limits, and no paywalls.

⚡
100% Free
No premium tiers, no hidden fees. All features are available to everyone by default.
🧠
Multi-Model
Switch between 18+ models from GitHub, Groq, and image generation providers.
🎨
Image Generation
Generate images using Pollinations.ai — free, no API key required, no limits.
🔒
Secure
All API keys are server-side only. Cloudflare Turnstile prevents bot abuse.
GETTING STARTED

Installation

LibreClaude runs on Vercel serverless functions. You can run it locally or deploy it to production in minutes.

Prerequisites

  • Node.js v18 or higher
  • Vercel CLI: npm install -g vercel
  • A Groq API key — console.groq.com
  • A GitHub Personal Access Token — github.com/settings/tokens
  • A Cloudflare Turnstile widget (free) — dash.cloudflare.com

Clone & Configure

bash 
git clone https://github.com/Fanatic911/LibreClaude.git
cd LibreClaude
cp .env.example .env

Open .env and fill in your API keys. See the Environment Variables section for details.

Run Locally

bash 
vercel dev

The app will be available at http://localhost:3000. Local mode automatically loads your .env file and writes structured JSON logs.

Image Generation

LibreClaude v1.1 introduces built-in image generation powered by Pollinations.ai — completely free with no API key required and no hard rate limits.

How It Works

Select an image model from the model picker. Type your prompt and send. The backend (api/image.js) builds a Pollinations URL with your prompt and parameters, returns it to the frontend, and the browser loads the image directly.

Available Image Models

ModelIDDescription
FluxfluxHigh quality, default choice
Flux Realismflux-realismPhotorealistic output
Flux Animeflux-animeAnime and illustration style
Flux 3Dflux-3d3D render aesthetic
TurboturboFaster generation, slightly lower quality
💡
Image generation uses img: prefixed model IDs internally (e.g. img:flux). The frontend detects this prefix and routes the request to /api/image instead of /api/chat.

Security

Cloudflare Turnstile

Every chat and image request requires a valid Turnstile token. The token is validated server-side before any API call is made. In local development (IS_LOCAL=TRUE), Turnstile validation is bypassed automatically.

Server-Side API Keys

All API keys (Groq, GitHub Token) live exclusively in environment variables and are never sent to the client. The frontend only receives AI-generated content — never credentials.

⚠️
Never commit your .env file. It is listed in .gitignore by default. Use the Vercel dashboard to set environment variables in production.

Local Logging

When running locally (IS_LOCAL=TRUE), LibreClaude writes structured JSON logs to help you debug and monitor activity. These files are excluded from Git.

FileContents
logs.txtAll chat requests — provider, model, status, response size
configlogs.txtConfig endpoint calls — Turnstile key presence, local flag
imagelogs.txtImage generation requests — model, prompt preview, status
PublishLogs.txtGit deployment history from the Publish.py GUI tool

Each log file keeps a rolling maximum of 50 entries. Older entries are dropped automatically.

API Endpoints

All endpoints live in the api/ folder and are automatically served as Vercel serverless functions at /api/*.

POST /api/chat

Sends a conversation to the selected text model and returns the AI response.

Request Body

json
{
  "messages":       [{ "role": "user", "content": "Hello" }],
  "model":          "openai/gpt-4o",
  "username":       "User",
  "turnstileToken": "..."
}

Response

json
{ "content": "Hello! How can I help you today?" }
POST /api/image

Returns a Pollinations.ai image URL for the given prompt and model.

Request Body

json
{
  "prompt":         "a golden sun over a dark forest",
  "model":          "flux",
  "width":          1024,
  "height":         1024,
  "turnstileToken": "..."
}

Response

json
{
  "url":    "https://image.pollinations.ai/prompt/...",
  "prompt": "a golden sun over a dark forest"
}
GET /api/config

Returns public configuration consumed by the frontend on load.

Response

json
{
  "turnstileSiteKey": "0x4AAA...",
  "isLocal": false
}
GET /api/links

Returns community links (Discord, GitHub) from environment variables.

Response

json
{
  "discord": "https://discord.gg/...",
  "github":  "https://github.com/..."
}

Deployment

Deploy to Vercel

bash
vercel --prod

Vercel automatically detects all files in the api/ folder as serverless functions. The outputDirectory: "." in vercel.json tells Vercel to serve index.html from the root.

Using Publish.py

LibreClaude ships with a local Git deployment GUI (Publish.py). Run it with Python to get a simple interface for committing and pushing to GitHub with version tagging. It automatically fixes .gitignore, purges sensitive files from Git cache, and logs all deployments to PublishLogs.txt.

bash
python Publish.py
